# CRACKL Privacy Policy Draft

This is a launch draft for legal review before public release.

## Data We Collect

- Account details: username, email address, avatar, city/area, and optional profile fields.
- Gameplay data: riddles served, answers submitted, solve status, time taken, hints used, XP, streaks, rankings, coins/Intels, rooms, challenges, and bounty attempts.
- Admin-created content: riddles, media URLs, layout configuration, and operational settings.
- Technical data: IP address, user agent, error logs, rate-limit events, and audit logs for security and abuse prevention.
- Payment/cashback data if enabled later: UPI ID, payout amount, status, and processing timestamps.

## How We Use Data

- Run the game, matchmaking, progression, rankings, wagers, and challenge links.
- Prevent duplicate riddles, cheating, brute force attempts, and exploit paths.
- Improve content quality, difficulty balance, reliability, and user experience.
- Operate admin tools, support, abuse review, and production monitoring.
- Process cashback or payout requests only if that system is enabled.

## Data Sharing

We do not sell personal data. Data may be processed by hosting, database, analytics, email, monitoring, and payment providers needed to operate CRACKL.

## Security

CRACKL uses backend-controlled database access, row-level security, admin audit logs, rate limits, and restricted public database permissions. No system is perfectly secure; report suspected issues immediately.

## User Rights

Users can request account deletion, data export, or correction by contacting support. Some records may be retained for fraud prevention, legal compliance, or audit integrity.

## Children

CRACKL should not be launched to underage users until age policy, consent requirements, and jurisdiction-specific rules are reviewed.
